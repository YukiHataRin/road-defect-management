請輸入六位小數的經度值

{
  "defects": [
    {
      "defect_type": 1,
      "latitude": 24.147736,
      "longitude": 120.673648,
      "district": "西屯區",
      "road_section": "文華路",
      "image_name": "crack_001.jpg"
    },
    ...
  ]
}
請選擇要上傳的圖片檔案，檔名需要與 JSON 中的 image_name 相符
